Installing Root6 on Ubuntu-type linux

1 Introduction

This short guide should get you set up for a relatively pain free root6 installation. It used cmake for the installation, which turns out to be quite convenient!

2 Prerequisites

  • Install from packages using this command: 
    sudo apt-get install <package_name>
    • emacs
    • build-essential
    • libgsl0-dev
    • libxerces-c-dev
    • libcoin80-dev
    • libglu1-mesa-dev
    • cmake
    • libexpat1-dev
    • libinventor1
    • magit
    • libxft-dev
    • libxml2-dev
    • libmotif-dev
    • libqt4-dev
    • libqt4-opengl-dev
    • tk

3 Preparation

  • I like to put things like geant4 and root in the /opt directory. It makes them system-wide and easier to separate from the other junk in your home folder.
  • You need to change the privileges to allow your user (longland in my case) 
    cd /opt
sudo mkdir root
chown -R longland:longland root

4 Download the root code

  • I'm using version 6.04.06
  • Get it here
  • Unzip. We'll put it in that opt directory we made earlier 
    cd /opt/root
mv downloaded-root-file.tar.gz .
tar -xzvf downloaded-root-file.tar.gz
  • This should have created a root-6.04.06 folder. I'll rename it: 
    mv root-6.04.06 root-6.04.06-src

5 Build

  • Here's how it'll work
    1. Make a build directory
    2. Use cmake to prepare the build directory
    3. Build root (compile it)
    4. Install the useful stuff in yet another folder - the install directory.
  • So: let's do steps 1 and 2 
    mkdir root-6.04.06-build/
cd root-6.04.06-build/
cmake -DCMAKE_INSTALL_PREFIX=/opt/root/root-6.04.06-install/\
      ../root-6.04.06-src
  • Note: This command told root to install the compiled code in our install directory (-DCMAKE_INSTALL_PREFIX=/opt/root/root-6.04.06-install) For more build options, see the root documentation
  • Almost there. Now's time to compile (step 3 above): 
    make -j6

    I'm compiling with 6 cores of my processor. You can choose less! Time for coffee and games!

  • Install the compiled code in the root-6.04.06-install directory (step 4 above) 
    make install
  • That's it! root is installed!

6 Wrapping Up

You just need to tell linux where to find root now. Root does the hard work for you here. Just put the following in your .bashrc file: 

source /opt/root/root-6.04.06-install/bin/thisroot.sh

Now log out, log back in, and get to work!
